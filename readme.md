# Roommates project

## Import project in Eclipse

1. Import -> Maven -> Existing Maven projects
2. Browse -> .. Finish

## Run project

1. Run -> Run configurations
2. Maven Build -> New launch configuration 
3. Name: roommates 
4. Base directory: ${workspace_loc:/Roommates} 
5. Goals: tomcat7:run
6. Profiles: pom.xml 
7. Run (this saves the configuration)
